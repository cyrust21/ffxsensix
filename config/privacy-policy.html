Privacy Policy

Last updated: February 02, 2026 • FFX SENSIX

FFX SENSIX berkomitmen untuk melindungi privasi Anda. Kebijakan ini menjelaskan bagaimana kami menangani data saat Anda menggunakan aplikasi kami.

1. Informasi yang Kami Kumpulkan

Kami meminimalkan pengumpulan data pribadi. Kami tidak mengumpulkan nama, alamat, atau daftar kontak. Data yang diproses meliputi:

  • Identifikasi Anonim: Autentikasi anonim untuk identitas pengguna unik.
  • Data Teknis: Jenis perangkat, versi OS, dan ID perangkat terenkripsi untuk validasi VIP.

2. Penggunaan API Shizuku

Aplikasi menggunakan API Shizuku untuk menjalankan optimasi sistem secara lokal:

  • Optimasi V-Sync untuk stabilitas frame rate.
  • Penyesuaian resolusi layar untuk responsivitas sentuhan.
  • Manajemen cache memori untuk menjaga performa perangkat.

Akses ini tidak mengambil file pribadi atau data sensitif pengguna.

3. Layanan Pihak Ketiga

Kami menggunakan layanan Google berikut untuk mendukung fungsionalitas aplikasi:

  • Firebase Authentication (Anonymous): Identitas unik anonim tanpa login manual.
  • Firebase Realtime Database: Penyimpanan konfigurasi dan validasi lisensi VIP.
  • AdMob: Penayangan iklan melalui pengumpulan data teknis perangkat.

Kebijakan Pihak Ketiga dapat diakses di: Google Privacy Services dan Firebase Security.

4. Keamanan Game & Data

FFX SENSIX bukan alat modifikasi (mod), cheat, atau hack. Tidak ada modifikasi kode biner pada aplikasi pihak ketiga. Semua data dikirim melalui protokol HTTPS terenkripsi.

5. Penyimpanan Data

Data anonim disimpan selama aplikasi digunakan. Data pada database kami akan dibersihkan secara otomatis jika tidak ada aktivitas selama 24 bulan.
